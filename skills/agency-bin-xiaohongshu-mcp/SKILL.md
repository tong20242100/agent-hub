# 🦞 小红书自动化安全操作手册 (Safe-Runbook)

## 🚨 绝对禁令 (Hard Rules)
1. **独占登录**: 严禁在运行 MCP 的同时，在任何电脑浏览器上登录同一账号。一旦发现多端网页登录，立即停止操作。
2. **频率熔断**: 
   - 每日发布上限：**50 篇**。
   - 连续操作间隔：必须大于 **30 秒**。
3. **内容去重**: 严禁直接上传未处理的原始网络图片。必须先调用图片处理工具进行裁剪或像素微调。

## 🛠️ 战术流程 (Tactical Workflow)
1. **感知**: 检查当前系统时间，若处于 01:00 - 07:00 (人类睡眠时间)，禁止发起自动发布任务。
2. **脱敏**: 检查文案中是否包含「微信」、「私聊」、「赚钱」等诱导词，若有则强制重写。
3. **闭环**: 每次发布成功后，记录在 `workspace/xhs_log.txt` 中，以监控每日限额。

## 🛡️ 证据等级 (L1 Evidence)
任何发布操作必须在日志中附带 `status: success` 的物理回执。
